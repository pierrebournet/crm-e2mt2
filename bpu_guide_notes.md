# BPU et SC v4 - Guide de renseignement (2ème fichier)

## Structure du fichier complet (Excel original)
Ce PDF est le guide de renseignement d'un fichier Excel qui contient les feuilles suivantes :

| Feuille | Contenu |
|---|---|
| BPU | Bordereau des Prix Unitaires |
| SC | Scénario de Commande (SC) |
| Récap SC et MG | Récapitulatif du Scénario de Commande et calcul du montant Minimum Garanti |
| **Détail D-Prest. connexes 1** | **Prix unitaires Mission D - Rubrique 1 : Prestations particulières** |
| **Détail D-Prest. connexes 2** | **Prix unitaires Mission D - Rubrique 2 : Taux horaires et coefficients de majoration** |
| **Détail D-Prest. connexes 3** | **Prix unitaires Mission D - Rubrique 3 : Coefficients pour l'achat de fournitures ou de sous-traitance** |
| Détail D-DQE | Détail Quantitatif Estimatif (DQE) : Simulation de commande mission D |

## Information clé
Les taux horaires (colonne D), les coefficients de majoration (colonnes F, H et I) et les coefficients d'achat fournitures/sous-traitance sont dans les feuilles "Détail D-Prest. connexes 1, 2 et 3".

Ce PDF ne contient que le guide, pas les données chiffrées. Le fichier source est un Excel protégé avec des cellules éditables sur fond bleu ciel.

## Conclusion
Il me faut le fichier Excel complet (ou les feuilles "Détail D-Prest. connexes 1, 2 et 3") pour vérifier les prix des devis. Le PDF guide seul ne contient pas les tarifs.
